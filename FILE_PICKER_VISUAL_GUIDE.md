| State              | Color | Hex Code | Usage                      |
|-------------------|-------|----------|----------------------------|

| File selected     | Green | #4CAF50  | Success/active state       |
| Error/Invalid     | Red   | #f44336  | Error state (if needed)    |

- Section spacing: 8px

## Integration Examples

### Example: Minimal Integration
```python
from src.ui.widgets import FilePickerWidget

picker = FilePickerWidget()
picker.files_selected.connect(on_files_selected)
layout.addWidget(picker)
```

### Example: Full Configuration
```python
picker = FilePickerWidget(
    file_types=('*.png', '*.jpg'),
    allow_multiple=True,
    allow_folders=True,
    allow_archives=True
)
picker.setToolTip("Select files to process")
```
