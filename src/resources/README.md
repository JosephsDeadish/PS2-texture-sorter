# Resource Folders

These folders will contain the application resources:

## icons/
- `panda_icon.ico` - Main application icon (for EXE)
- Tab icons
- UI element icons

## cursors/
- `skull_cursor.cur` - Skull cursor
- `panda_cursor.cur` - Panda cursor  
- `sword_cursor.cur` - Sword cursor

## themes/
- Color theme files (.json)
- Pre-built theme definitions

## sounds/
- Notification sounds
- Completion alerts
- Custom sound effects

## Creating Icons

For the panda icon, you can:
1. Create a `.ico` file with panda design
2. Use online tools like: https://convertio.co/png-ico/
3. Recommended size: 256x256 px
4. Place at: `src/resources/icons/panda_icon.ico`

## Note

The application will work without custom resources by using defaults.
Custom resources enhance the panda-themed experience.
